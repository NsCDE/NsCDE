7.25. WSM.conf

Non-FVWM config file. Here, names of the desks are provided and can be changed manually of with Workspace Manager. Names of the desk pages (for the menus) are defined here too, and manager manually only for now. Option WSPPG can have value 0 or 1. If 1, small dynamic icon similar to the PageIndicator will be shown right of the desk name on the Workspace Manager active desk button, but only in 2 and 4 button configuration. Off by default, edit manually.
