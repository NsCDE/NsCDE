4.9. Occupy Page

This tool dialog is called from the left titlebar button menu. Or from Alt+Space key combination from the window context. It is exact copy of the same CDE dialog and it sends selected window to named page selected from the list, or it can make it sticky across pages of the current workspace with All Pages checkbox pressed in. One addition here is the checkbox Go With the Window; when checked, makes NsCDE to change a current page and go where window was sent. OK performs an action, Dismiss quits Occupy Page tool.

Key Bindings:

  • Escape: Quits Occupy Page without performing an action.

  • H: Displays this help text, like Help was pressed.

  • A: Checks All Pagess checkbox.

  • Return: Performs move action like OK button was pressed, without going to the selected page.

  • Ctrl+Return: Performs move action like OK button was pressed, and changes active page to the same destination.

  • Up/Down: Selects page in the up or down direction on the list.

  • Sun Help and F1: Displays this help text.
