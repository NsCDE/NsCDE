7.3. NsCDE-Animate.conf

FVWM Animate Module configuration. Animate module is started by NsCDE by default automatically, but with None as a default effect. This can be reconfigured by the user in private $FVWM_USERDIR/NsCDE-Animate.conf (or .local) with editor or with Window Style Manager. None effect is choosen as default for increased CDE similarity, because CDE doesn't have iconification animation effects.
