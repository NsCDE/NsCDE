4.5. MonthDayApplet

Familiar CDE icon with month and day of the month in it. Simple applet which calls empty, do-nothing (by default) function f_Calendar. This function can be overridden in $FVWM_USERDIR/Functions.fvwmlocal to call a program configured by the user.

If Sun type keyboard is in use, Help key above Month Day Applet will bring this help text in browser. On PC keyboard, F1 has the same function.
