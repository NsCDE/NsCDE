5.17. Sysinfo

Well known Workstation Information info dialog. It doesn't have any functions apart close (Dismiss) button. It's role is informational. It displays current username, hostname, machine and CPU architecture type. IP address, hostid, network (NIS, NISPLUS, LDAP ...) domain name, internet domain name, size of the physical RAM, swap size, swap usage (as progressbar), operating system long name, and then version of the FVWM and version of the NsCDE. Last, it shows time when system was last booted. This dialog can be found on the Tools or fourth Subpanel of the Front Panel in default configuration, under the entry Workstation Info. In the context of it's window, keys Escape, Return and Q will close a window, while Sun Help and F1 displays this documentation.
