9.14. Ident.fvwmconf

Module FvwmIdent(1) is called either from a small menu which can be popped up with middle pointer click on a titlebar, or from the root window version of the Window Options menu. This is FvwmIdent's configuration file. It simply defines colorset and font for the FvwmIdent's module window.
