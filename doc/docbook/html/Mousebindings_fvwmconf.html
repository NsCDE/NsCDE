9.18. Mousebindings.fvwmconf

File Mousebindings.fvwmconf is made in similar manner as the Keybindings.<kbd_bind_set>. Can be overridden (.fvwmconf) or extended (.fvwmlocal) just like (almost) any NsCDE conf file. See fvwm(1) for "Mouse" commands. Here, commands invoked with pointer are defined. As it is the case with a Key Bindings, there are considerable more mouse actions in NsCDE than in original CDE. The syntax is described at the top of the file, similar to this of Key Bindings.

Default set of mouse bindings in NsCDE is described in the chapter about Mouse Bindings.
