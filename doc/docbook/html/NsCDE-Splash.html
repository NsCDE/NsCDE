6.6. Splash - FvwmScript

Figure 60. NsCDE Splash Screen


Splash screen is started at the beggining of the NsCDE session. It covers all the screen(s) and displays NsCDE logo in upper left corner, and on the lower left corner there is a info about system on which NsCDE is running, FVWM version used, licence, and startup message. Pressing Ctrl+C or clicking at the text exits Splash screen before it's timeout, which is 7-8 seconds. Splash is wrapped in the f_Splash function and can be called interactively during existing session by calling this function.
